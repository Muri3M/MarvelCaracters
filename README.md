# Marvel Caracters


Instalando Marvel Caracters

Para instalar o Marvel Caracters list basta rodar o comando:

  ```sh
  yarn install
  ```

Para rodar o Music List em modo de desenvolvimento basta rodar o comando:

  ```sh
  yarn androi ou yarn ios
  ```
